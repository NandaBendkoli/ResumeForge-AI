# 🚀 ResumeForge-AI

An AI-powered Interview Preparation & Resume Intelligence Platform built using the MERN Stack and Google Gemini AI.

ResumeForge-AI helps students, freshers, and job seekers prepare smarter for interviews by generating personalized interview reports, skill-gap analysis, ATS-focused insights, and structured preparation roadmaps using Generative AI.

# 📌 Features

## 🔐 Authentication System
- JWT Authentication
- Protected Routes
- Secure Login & Registration
- HTTP-only Cookie Support
- Context API Authentication State Management

---

## 📄 Resume Upload & Parsing
- Upload Resume PDF
- Extract Resume Content
- Analyze Skills & Experience
- Compare Resume Against Job Description

---

## 🤖 AI-Powered Interview Report Generation
Using Google Gemini AI, the system generates:

- ATS Match Score
- Technical Interview Questions
- HR & Behavioral Questions
- Interview Intentions
- Ideal Answers
- Skill Gap Analysis
- Personalized Preparation Roadmap

## 📊 Smart Skill Analysis
The platform intelligently identifies:

- Missing Skills
- Weak Technologies
- Knowledge Gaps
- High Priority Topics

## 🛣 Personalized Preparation Plan

- Daily Preparation Tasks
- Topic Revision Plans
